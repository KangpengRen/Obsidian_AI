分类问题最常用的损失函数：**交叉熵误差（Cross Entropy Error）**：
$$
L = -\frac{1}{n} \sum_{i=1}^n t_i log y_i
$$
其中，$log$表示自然对数，$y_i$表示神经网络的输出，$t_i$表示正确解标签；而且，$t_i$中只有正确解标签的值为$1$，其它均为$0$。

代码实现：
```python
def cross_entropy_error(y, t):
    """
    交叉熵误差
    """
    # 如果y是1维的，转化为2维
    if y.ndim == 1:
        t = t.reshape(1, t.size)
        y = y.reshape(1, y.size)
    # 监督数据是one-hot编码的情况下，转换为正确标签的索引
    if t.size == y.size:
        t = t.argmax(axis=1)
    batch_size = y.shape[0]
    return -np.sum(np.log(y[np.arange(batch_size), t] + 1e-7)) / batch_size
```
